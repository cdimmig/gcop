\subsection{\-Detailed \-Description}
\-A \char`\"{}geometric\char`\"{} planar rigid body system. \-The control system is implemented as an evolution on \-S\-E(2) as opposed to a vector space.

\-The state is $ \bm x = (g, v) $ where $ g\in SE(2)$ is the pose, and $ v\in \mathbb{R}^3$ is the velocity
