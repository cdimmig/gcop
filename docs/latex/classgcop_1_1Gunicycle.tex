\subsection{\-Detailed \-Description}
\-A \char`\"{}geometric\char`\"{} unicycle model with 2nd order dynamics. \-This is often used to model simple wheeled robots or fixed-\/wing aircraft. \-The control system is implemented as an evolution on \-S\-E(2) as opposed to a vector space.

\-The state is $ \bm x = (g, \omega, v) $ where $ g\in SE(2)$ is the pose, the controls $ \bm u = (u_\omega,u_v) \in\mathbb{R}^2$ correspond to turn rate acceleration and forward acceleration.
