\subsection{\-Detailed \-Description}
\-A simple rear-\/drive \doxyref{\-Rccar}{p.}{classgcop_1_1Rccar} model derived from \-Vehicle \-Demo using \-Bullet \-Physics \-Engine

\-This car model uses the full 6\-D\-O\-F state for simulation. \-To keep compatibility with existing 2\-D rccar model, this class only provides the state output as $ \bm x = (x,y,\theta, v) \in\mathbb{R}^4$. \-The controls for the car are given as desired steering angle and desired body velocity. \-These controls are converted into true controls for the bullet rccar i.\-e driving torque and actual steering angle through proportional control. \-The gains for the controller are input as parameters to the system \#\-T\-O\-D\-O \-Use full state of the car
